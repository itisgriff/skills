# Data Mutation

All writes happen inside a `database.write` transaction block.

## Insert

Use the `Draft` type (generated by `@Table`) for inserts:

```swift
try database.write { db in
    try Item.insert { Item.Draft(name: "New Item", isComplete: false) }.execute(db)
}
```

When you supply the primary key:

```swift
try database.write { db in
    try Item.insert { Item(id: UUID(), name: "New Item", isComplete: false) }.execute(db)
}
```

## Update

```swift
try database.write { db in
    try Item
        .where { $0.id == itemID }
        .update { $0.name = "Updated Name" }
        .execute(db)
}
```

Update multiple columns:

```swift
try database.write { db in
    try Item
        .where { $0.id == itemID }
        .update {
            $0.name = "Updated"
            $0.isComplete = true
        }
        .execute(db)
}
```

## Delete

```swift
try database.write { db in
    try Item.where { $0.id == itemID }.delete().execute(db)
}

// Delete multiple:
try database.write { db in
    try Item.where { $0.id.in(ids) }.delete().execute(db)
}

// Delete all:
try database.write { db in
    try Item.all.delete().execute(db)
}
```

## Transactions

All operations inside `database.write { db in ... }` run in a single transaction. Group related operations together:

```swift
try database.write { db in
    try Item.insert { Item(id: itemID, name: "New") }.execute(db)
    try Attendee.insert { Attendee.Draft(name: "Alice", itemID: itemID) }.execute(db)
    try Attendee.insert { Attendee.Draft(name: "Bob", itemID: itemID) }.execute(db)
}
```

## Seeding Data

Use `Database.seed` to insert initial data. It inserts rows in batches of consecutive records of the same table:

```swift
try db.seed {
    Item(id: UUID(), name: "Design Review", isComplete: false)
    Item(id: UUID(), name: "Sprint Planning", isComplete: false)
    for name in ["Alice", "Bob", "Charlie"] {
        Attendee.Draft(name: name, itemID: someItemID)
    }
}
```

`seed` uses the `@SeedsBuilder` result builder, which supports `for` loops, `if`/`else`, and other standard result builder syntax.
